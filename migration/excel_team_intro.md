## ⏱️ Credits & Token Usage — Please Read!

Running an AI assistant costs credits (processing tokens). Here's how to be efficient:

### ✂️ Use `/new` to start fresh sessions often
- Every conversation uses a "context window" — the longer the conversation, the more credits each message costs
- When the context fills up (Sisi will warn you), **type `/new`** to start a fresh session
- Rule of thumb: `/new` after completing a task, or when Sisi warns you context is getting high
- **Example:** We burned 20,000+ credits in one session once — `/new` more often prevents that

### 📌 Pin important information before `/new`
- Say **"pin it"** to save a snapshot of where you are before starting fresh
- This saves: what we were working on, key decisions, next steps, files involved
- To restore: say **"bring back [topic]"** in the new session
- ⚠️ AI memory doesn't survive session resets — if you don't pin it, it's gone!

### 💡 Context size warning
Sisi MaxClaw will warn you:
- **< 25% full:** All good, no mention
- **25–50% full:** ⚠️ "Consider /new after this topic"
- **> 50% full:** 🔴 "Recommend /new soon"
