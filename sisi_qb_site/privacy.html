SisiBookkeeper — Privacy Policy

Effective Date: March 16, 2026

This Privacy Policy describes how SisiBookkeeper ("the Application") handles data accessed through the QuickBooks Online API.

1. Scope

SisiBookkeeper is a private internal application used exclusively by Excel Plumbing Supply + Showroom. It is not a public application and has no end users outside of the Company.

2. Data We Access

The Application accesses: Bank accounts and transaction data, Vendor bills and payments, Financial reports and account balances, Chart of accounts.

3. How We Use Data

Data is used solely for internal bookkeeping and bank reconciliation purposes. No data is sold, shared, or disclosed to any third party.

4. Data Storage

Data accessed through the API is processed in real-time for bookkeeping operations. No financial data is permanently stored outside of QuickBooks Online.

5. Data Security

All API communications are conducted over HTTPS. OAuth 2.0 tokens are stored securely and never shared externally.

6. Your Rights

Authorization can be revoked at any time through QuickBooks Online → Settings → Authorized Apps.
